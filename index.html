Municipal Ticket System (Programming POE)

A WPF application built using C# and SQL Server, designed to streamline municipal issue reporting.

The application allows users to report issues, track their status, and stay updated on local events. It features an AI chatbot, powered by the Ollama API, to assist users with queries. The system utilizes data structures like queues, linked lists, graphs, and binary search trees for efficient processing.

Users can submit reports by selecting a location, choosing a category, adding a description, and optionally uploading a file.
